# Carteira Financeira - Simulador RESTful

## Descrição
Este projeto é uma aplicação RESTful desenvolvida em Java utilizando Spring Boot, com o objetivo de simular uma carteira financeira. A aplicação permite aos usuários realizar operações como cadastro, depósito, saque e consulta de saldo.

## Pre-requisitos

* **Sistema Operacional:** Windows, macOS ou Linux com Docker instalado.
* **Docker:** Versão 27.3.1 ou superior.
* **Gradle:** Para gerenciar as dependências do projeto.
* **Java:** 17 ou superior 
* **Banco de dados relacional:** Configure o arquivo application.properties com as credenciais do seu banco de dados.
